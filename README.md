## Run Locally

Clone the project

```bash
  git clone https://github.com/businesspages29/vindaloo
```

Go to the project directory

```bash
  cd vindaloo
```
Install dependencies using composer 

```bash
  composer install
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm build
```

Setup Migrations with seeder

```bash
  php artisan migrate --seed
```
